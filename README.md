# افزونه شناورک (Shnavork)

افزونه شناورک یک دکمه شناور برای نمایش راه‌های ارتباطی در سایت وردپرسی است.

## امکانات

- دکمه شناور زیبا
- پشتیبانی از:
  - واتساپ
  - تلگرام
  - اینستاگرام
  - تماس تلفنی
- تنظیم رنگ دکمه
- فعال/غیرفعال کردن هر شبکه اجتماعی

## نصب

1. فایل zip افزونه را دانلود کنید
2. در وردپرس بروید به:

افزونه‌ها → افزودن → بارگذاری افزونه

3. فایل zip را آپلود کنید
4. افزونه را فعال کنید

## تنظیمات

بعد از فعالسازی از منوی:

تنظیمات → شناورک
